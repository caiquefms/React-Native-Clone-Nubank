## Nubank Layout Clone

Construção de uma aplicação que simula a tela principal do aplicativo Nubank, fazendo uso de bibliotecas de terceiros para a utilização de Qrcode, manipulação do touch, entre outros.

```Bibliotecas Utilizadas
    react-native-gesture-handler
    react-native-qrcode
    react-native-vector-icons
    react-navigation
    styled-components	
```
